## Overview

This project is an interactive **Power BI dashboard** built to analyze e-commerce sales performance across **cities, categories, months, customers, payment modes, and sub-categories**.

The report turns raw transactional data into a clean business story with visuals for:
- sales performance by city
- quantity distribution by category
- monthly profit trends
- top customer contribution
- payment mode usage
- profit by sub-category
- year-wise filtering

The dashboard is designed for portfolio presentation and demonstrates practical skills in **data cleaning, modeling, visualization, and business reporting**.

---

## Business Goals

The dashboard was built to answer questions such as:
- Which cities generate the highest sales?
- Which product categories drive the most quantity sold?
- How does profit change month by month?
- Which customers contribute the most revenue?
- Which payment method is used most often?
- Which sub-categories are the most profitable?

## Key Dashboard Features

- **Year-wise slicer** for 2023, 2024, and 2025

- **Quantity by Category** donut chart
- **Profit by Month** column chart
- **Sales by Customer Name** bar chart
- **Quantity by Payment Mode** donut chart

## Dashboard Insights

From the current dashboard analysis:

- **Cities such as Surat, Patna, Ranchi, and Pune** contribute strongly to sales.
- **Monthly profit** stays fairly stable, with some visible peaks across the year.
- **Category mix** is well diversified across furniture, books, kitchen, home decor, clothing, sports, toys, and groceries.
- **Payment modes** are balanced across COD, net banking, credit card, debit card, and UPI.
- **Top customers** drive a significant portion of revenue, showing the value of customer-level analysis.
- **Sub-categories** like sofa, lamp, tennis racket, non-fiction, and headphones contribute meaningfully to profit.

## Dataset Structure

The Power BI model uses a dataset named:

`Ecommerce_Sales_Data_2024_2025`

Typical fields used in the report include:

| Column | Purpose |
|---|---|
| Order Date | Used for monthly and yearly analysis |
| Sales | Core revenue metric |
| Profit | Profit trend analysis |
| Quantity | Quantity sold analysis |
| City | Geographic sales performance |
| Category | Product grouping |
| Sub-Category | Detailed product performance |
| Customer Name | Customer-level sales analysis |
| Payment Mode | Payment behavior analysis |

## How to Use

1. Download or clone this repository
2. Open the `.pbix` file in **Power BI Desktop**
3. Refresh the dataset if required
4. Use the **Year slicer** to filter between 2023, 2024, and 2025
5. Explore the visuals to analyze sales, profit, customers, and categories
